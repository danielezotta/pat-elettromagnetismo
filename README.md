Android app for browsing electromagnetic permits from PAT (Provincia Autonoma di Trento) and monitoring APPA's Sistema Informativo Misure Campi Elettromagnetici. Features interactive map visualization of cellular towers, radio/TV transmitters, and permit documents.

### Features
- Interactive APPA electromagnetic field monitoring map
- Browse PAT permit registry with search
- Document viewer for permits and attachments
- Periodic notifications for new permits
- Cell tower (SRB) and broadcast transmitter locations

### Technologies
- API 35 with Jetpack Compose
- OpenStreetMap integration
- PAT & APPA official data sources
- AI-assisted features (v1.2.0+)
